##需求描述：

**购物车：**

* 商品信息- 数量、单价、名称

* 用户信息- 帐号、密码、余额

* 用户可充值

* 购物历史信息

* 允许用户多次购买，每次可购买多件

* 余额不足时进行提醒

* 用户退出时 ，输出档次购物信息

* 用户下次登陆时可查看购物历史

* 商品列表分级显示


##脚本执行方式：

* 调用python 执行 shoping_mall.py ，python shoping_mall.py
* 运行环境： python 3.x
